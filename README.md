# Agent Craftsmanship

> Formation 12 semaines pour passer de "expert logiciel + IA" à "architecte de systèmes agentiques crédible et différenciant"

## Positionnement

```
"L'IA génère le code, l'IA génère les interfaces.
 Moi, je construis les rails de sécurité (Architecture & TDD)
 et je structure la donnée (DDD) pour que le train de l'IA
 ne déraille pas à pleine vitesse."
```

## Le parcours

| Phase | Semaines | Objectif |
|-------|----------|----------|
| **Phase 1** | 1-3 | Déprogrammer le réflexe "interface" — Penser IA comme système |
| **Phase 2** | 4-7 | Orchestration agentique réelle — Construire des systèmes multi-agents |
| **Phase 3** | 8-12 | Positionnement marché — Rendre l'expertise visible et monétisable |

## Mapping Craft → Agent Craft

| Compétence Craft | Application IA |
|------------------|----------------|
| **DDD** | Context Boundaries des Agents |
| **TDD** | LLM Evals (Ragas, DeepEval) |
| **Legacy Refactoring** | Data Engineering pour RAG |
| **Clean Architecture** | Séparation agents métier / infra |

## Projet Signature : "Le Refactor Automatisé"

Système multi-agents (Scanner → Architecte → Reviewer → Coder) qui démontre la valeur "AI Reliability Architect".
